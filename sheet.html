HTML Cheat Sheet

1. Headers & Paragraphs

  • <h1> to <h6>: Headings (largest to smallest)
  • <p>: Paragraph of text

2. Text Formatting

  • <b> or <strong>: Bold text
  • <i> or <em>: Italic text
  • <mark>: Highlighted text
  • <br>: Line break
  • <hr>: Horizontal rule
  • <code>: Inline code

3. Lists

  • <ul>: Unordered list (bullets)
  • <ol>: Ordered list (numbers)
  • <li>: List item

4. Images

  • <img src="url" alt="desc">: Displays an image

5. Hyperlinks

  • <a href="url">Text</a>: Clickable link
  • target="_blank": Open in new tab

6. Tables

  • <table>: Table container
  • <tr>: Table row
  • <th>: Table heading
  • <td>: Table data cell

7. Iframes

  • <iframe src="url"></iframe>: Embed webpage or content

8. Videos

  • <video src="video.mp4" controls></video>: Embed a video
  • Use <source> inside for multiple formats

9. Anchor Tags

  • <a href="#section">: Jump to ID
  • <div id="section">: Target section

10. Divs

  • <div>: Generic container (block element)
  • Used for layout, grouping, and styling

Bonus Tips

  • All tags use <> and most have an opening and closing tag
  • Use proper nesting and indentation
  • Use alt attributes in images for accessibility
  • Use semantic HTML like <header>, <nav>, <footer>
